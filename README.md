# Quiz-Game-System

 ## 1. Introduction
The Quiz Game System is a simple C++ based application that allows users to answer multiple-choice questions and calculate their score. This system is designed to test the user’s knowledge of basic programming concepts.
It is developed using Object-Oriented Programming (OOP) concepts such as classes and functions, making the system structured and easy to understand.

## 2. Objective
<li>To understand Object-Oriented Programming in C++
<li>To develop a quiz-based application
<li>To test user knowledge using multiple-choice questions
<li>To calculate and display final score
<li>To improve logical and programming skills
  
## 3. System Design
The system uses:
<li>Class for managing quiz data
<li>Functions for each question
<li>Function to start and control the quiz
<li>Variable to store and update score

## 4. Conclusion
The Quiz Game System successfully demonstrates how C++ can be used to create an interactive application. It uses classes and functions to manage questions and calculate scores efficiently.
This project helps in understanding user input handling, conditional statements, and object-oriented programming concepts. It can be further enhanced by adding more questions, difficulty levels, and timer-based quizzes.
